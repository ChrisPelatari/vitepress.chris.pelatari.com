# Resurrection of an hp all in one

I still have to upgrade some hardware on it. It has an aging one terabyte drive. And I'm going to max out the ram at 16 GB. (I would try to max it out to 32 GB, but it's questionable whether it would work or not.)

To be fair, I had installed Linux mx a long time ago. it's Debian based and works really nice with the touchscreen.

When I booted it up, it only had 4 GB of RAM. I changed the host name. I installed Claude code. I immediately thought I want to interact with this over Signal. so I had Claude build me a small python daemon that I call [signal-claw](https://github.com/ChrisPelatari/signal-claw)
