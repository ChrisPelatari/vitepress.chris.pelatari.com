# Memories context and expectations 

I have the least surprising confession: I spent a little time on the GitHub repo for the journal-this skill and I fell into the familiar trap: I had started a new session without saving enough memory from the first turn and prompted the wrong model for a small task as if it would magically know what I was talking about. 

> I b0rked the install of my own agent skill. 

I had been using it for a couple of weeks already but yep. It’s still hard to remember the context **you** remember doesn’t always match what the agent is working with. I find it happens after a win or if I’m on a roll. If you approach any agent session with a lack of focus or simply aren't paying enough attention, they will mirror you and behave in unexpected ways. So despite releasing a journal skill this morning, I thought I’ll just journal it the old fashioned way. Heh.
