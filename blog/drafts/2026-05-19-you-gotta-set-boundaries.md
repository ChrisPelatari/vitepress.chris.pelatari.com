# you gotta set boundaries

I was doing scrolling after heated session with Claude code last night. I let myself get too emotional about the fact that AI had helped me figure out how to decompile a  [winforms blog posting client written by me in 2004](blog/posts/2004-06-03-Starting-a-conversation-I-want-to-hear-your-opinion.md).

This is a story about how I realized that I lost control of a situation with a software robot that I could’ve stopped at any time.  It led to having one of the best sessions today because I set boundaries at the very beginning.
